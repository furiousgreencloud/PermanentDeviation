THINGS TO KNOW:

Copy and Paste is disabled. You can only delete 39 characters at a time.

If you get 50 points then you can clear all and "start fresh".

The "username" is only used to allow you to collect points. It is connected to a cookie in your browser. If you clear cookies in your browser then you clear your points.

You don't have to code. You can either watch live coding (if someone is already coding) or you can REPLAY the last code written.

POINTS / SCORING:

- Points are secured on a successive "Play", and are lost if code does not compile.
- Submit has no special effect on points, except that it is similar to clicking "Play"
- Comments = 2 points
- Number = 1 point
- Keyword = 2 points
- Processing words = 1 point
- White space / removing code = 0 point
